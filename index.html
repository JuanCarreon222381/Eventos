Ideas Principales

  • El Html DOM le permite ejecutar codigo cuando ocurre un evento.
  • El navegador genera eventos cuando "sucede cosas" a elementos HTML como por ejemplo:
  • 1. Al hacer click con el mouse: onclick
  • 2. Al colocar el mouse sobre un elemento: onmouseover
  • 3. Al alejar el mouse de un elemento : onmouseout
  • 4. Cuando una pagina web se ha cargado : onload
  • 5. Al presionar una tecla onkeydown
  • 6. Al hacer scroll en la pagina onscroll

Resumen


  • EvenListener:


    El metodo addEventListener() adjuta un controlador de evetnos al elemento especificado.

  • La sintaxis para este metodo es la siguiente:


    Elemento.addEventeListener(evento, fucnion).

  • El metodo removeEvenListener()


    se usa para adjuntar un cotrolador de eventos al documento HTML.
